## List All Agents

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8001/api/v1/agents \
  -H "X-API-Key: your-api-key"
```

```python Python
response = requests.get(
    "http://localhost:8001/api/v1/agents",
    headers={"X-API-Key": "your-api-key"}
)
```

</CodeGroup>

### Response

```json 200 Success
{
  "agents": [
    {
      "name": "researcher",
      "role": "Research and analyze information",
      "llm": "claude",
      "status": "active",
      "memory": true,
      "tools": ["web-search", "wikipedia"]
    }
  ]
}
```

## Get Agent Status

<CodeGroup>

```bash cURL
curl -X GET http://localhost:8001/api/v1/agents/researcher/status \
  -H "X-API-Key: your-api-key"
```

</CodeGroup>

### Response

```json 200 Success
{
  "name": "researcher",
  "status": "active",
  "currentTask": "Research AI trends",
  "tasksCompleted": 42,
  "uptime": "3d 2h 15m"
}
```

---

## Restart Agent

<CodeGroup>

```bash cURL
curl -X POST http://localhost:8001/api/v1/agents/researcher/restart \
  -H "X-API-Key: your-api-key"
```

</CodeGroup>
