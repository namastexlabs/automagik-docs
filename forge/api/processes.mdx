## Overview

Processes represent the actual execution of attempts. Monitor running AI agents, check resource usage, and control execution.

**Base URL**: `/api/processes`

---

## List Processes

Get all active processes.

```http
GET /api/processes
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | enum | Filter by status (running, completed, failed) |
| `llm` | string | Filter by AI agent |

### Example Request

```bash
curl http://localhost:8080/api/processes?status=running
```

### Example Response

```json
{
  "success": true,
  "data": [
    {
      "id": "proc_abc123",
      "attemptId": "attempt_xyz789",
      "taskId": "task_def456",
      "llm": "claude",
      "status": "running",
      "startedAt": "2024-01-15T10:30:00Z",
      "duration": 45000,
      "resources": {
        "cpuUsage": 45.2,
        "memoryUsage": 512000000,
        "diskIO": 1024000
      },
      "progress": {
        "current": "Generating authentication logic",
        "percentage": 65
      }
    }
  ]
}
```

## Cancel Process

Stop a running process.

```http
POST /api/processes/:id/cancel
```

### Example Request

```bash
curl -X POST http://localhost:8080/api/processes/proc_abc123/cancel
```

### Example Response

```json
{
  "success": true,
  "data": {
    "id": "proc_abc123",
    "status": "cancelled",
    "cancelledAt": "2024-01-15T10:32:00Z"
  }
}
```

## Get Process Logs

Stream real-time logs from a process.

```http
GET /api/processes/:id/logs
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `follow` | boolean | Stream logs in real-time |
| `tail` | integer | Number of recent lines to show |

### Example Request

```bash
curl "http://localhost:8080/api/processes/proc_abc123/logs?follow=true"
```

## SDK Examples

### JavaScript/TypeScript

```typescript
import { ForgeClient } from '@automagik/forge-sdk';

const forge = new ForgeClient();

// List running processes
const processes = await forge.processes.list({
  status: 'running'
});

// Get process details
const process = await forge.processes.get('proc_abc123');

// Monitor progress
const interval = setInterval(async () => {
  const proc = await forge.processes.get('proc_abc123');
  console.log(`Progress: ${proc.progress.percentage}%`);

  if (proc.status !== 'running') {
    clearInterval(interval);
  }
}, 1000);

// Stream logs
const logs = await forge.processes.logs('proc_abc123', {
  follow: true
});

for await (const log of logs) {
  console.log(log);
}

// Cancel if needed
await forge.processes.cancel('proc_abc123');
```
