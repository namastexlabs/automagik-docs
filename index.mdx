## ğŸ¯ Why Automagik?

### The Problem We're Solving

AI development today is **fragmented and complex**:

<AccordionGroup>
  <Accordion title="Lost in chat histories">
    No persistent task management. Context gets lost between conversations.
  </Accordion>
  <Accordion title="Vendor lock-in">
    Stuck with one AI provider. Can't compare or switch models easily.
  </Accordion>
  <Accordion title="Production barriers">
    Weeks to deploy simple agents. Complex infrastructure requirements.
  </Accordion>
  <Accordion title="Manual everything">
    AI that only responds, never acts. No proactive automation.
  </Accordion>
  <Accordion title="Integration hell">
    Every API requires custom code. No standard protocols.
  </Accordion>
</AccordionGroup>

### Our Solution: A Complete Automation Toolkit

<Steps>
  <Step title="Genie orchestrates AI agents">
    Ships ready-to-run AI workspace across any codebase with deep repository understanding
  </Step>
  <Step title="Forge manages workflows">
    Visual Kanban board with Git worktree isolation for safe experimentation
  </Step>
  <Step title="Hive executes multi-agent teams">
    YAML-based agent creation with persistent memory and RAG built-in
  </Step>
  <Step title="Omni connects to channels">
    Multi-tenant messaging hub for WhatsApp, Discord, Slack, and more
  </Step>
  <Step title="Spark schedules automation">
    Transform reactive agents into proactive 24/7 autonomous workers
  </Step>
  <Step title="Tools bridges APIs">
    Auto-generate MCP tools to connect any API to any AI instantly
  </Step>
</Steps>
