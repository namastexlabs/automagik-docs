## List Message Traces

Get message traces with filtering and pagination.

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8000/api/v1/traces?limit=10" \
  -H "X-API-Key: your-api-key"
```

```python Python
response = requests.get(
    "http://localhost:8000/api/v1/traces",
    headers={"X-API-Key": "your-api-key"},
    params={"limit": 10, "instance": "my-whatsapp"}
)
```

</CodeGroup>

### Query Parameters

<ParamField query="instance" type="string">
  Filter by instance name
</ParamField>

<ParamField query="phone" type="string">
  Filter by phone number
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `sent`, `delivered`, `read`, `failed`
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of traces to return (max 500)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of traces to skip
</ParamField>

### Response

```json 200 Success
{
  "traces": [
    {
      "id": "trace_abc123",
      "messageId": "3EB0ABC123...",
      "instance": "my-whatsapp",
      "to": "+1234567890",
      "message": "Hello!",
      "status": "delivered",
      "sentAt": "2025-10-31T12:00:00Z",
      "deliveredAt": "2025-10-31T12:00:05Z"
    }
  ],
  "total": 150,
  "limit": 10,
  "offset": 0
}
```

## Get Specific Trace

<CodeGroup>

```bash cURL

  -H "X-API-Key: your-api-key"
```
