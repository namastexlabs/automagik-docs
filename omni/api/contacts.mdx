## Overview

Access and manage contacts across all messaging platforms through a unified interface. The Omni Contacts API provides consistent contact data regardless of the underlying channel (WhatsApp, Discord, etc.).

<Note>
Contact endpoints are available NOW via the `/api/v1/omni/{instance_name}/contacts` path.
</Note>

---

## List Contacts

Retrieve all contacts for an instance with filtering and pagination.

**Endpoint**: `GET /api/v1/omni/{instance_name}/contacts`

<ParamField path="instance_name" type="string" required>
  Name of the instance
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number (1-based)
</ParamField>

<ParamField query="page_size" type="integer" default="50">
  Items per page (max: 500)
</ParamField>

<ParamField query="search_query" type="string">
  Search contacts by name
</ParamField>

<ParamField query="status_filter" type="string">
  Filter by contact status
</ParamField>

<ParamField query="channel_type" type="string">
  Filter by channel type (`whatsapp`, `discord`)
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8882/api/v1/omni/my-whatsapp/contacts?page=1&page_size=50" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "http://localhost:8882/api/v1/omni/my-whatsapp/contacts",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"page": 1, "page_size": 50, "search_query": "John"}
)

data = response.json()
for contact in data["contacts"]:
    print(f"{contact['display_name']}: {contact['phone_number']}")
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:8882/api/v1/omni/my-whatsapp/contacts?page=1&page_size=50',
  {
    headers: {
      'x-api-key': 'YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(data.contacts);
```

## Get Contact by ID

Retrieve detailed information about a specific contact.

**Endpoint**: `GET /api/v1/omni/{instance_name}/contacts/{contact_id}`

<ParamField path="instance_name" type="string" required>
  Name of the instance
</ParamField>

<ParamField path="contact_id" type="string" required>
  Contact identifier (e.g., `5511999999999@s.whatsapp.net` for WhatsApp, Discord user ID for Discord)
</ParamField>

<CodeGroup>

```bash cURL
curl -X GET "http://localhost:8882/api/v1/omni/my-whatsapp/contacts/5511999999999@s.whatsapp.net" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

contact_id = "5511999999999@s.whatsapp.net"
response = requests.get(
    f"http://localhost:8882/api/v1/omni/my-whatsapp/contacts/{contact_id}",
    headers={"x-api-key": "YOUR_API_KEY"}
)

contact = response.json()
print(f"Contact: {contact['display_name']}")
```

```javascript JavaScript
const contactId = '5511999999999@s.whatsapp.net';
const response = await fetch(
  `http://localhost:8882/api/v1/omni/my-whatsapp/contacts/${encodeURIComponent(contactId)}`,
  {
    headers: {
      'x-api-key': 'YOUR_API_KEY'
    }
  }
);

const contact = await response.json();
console.log(contact);
```

</CodeGroup>

## Use Cases

### List and Search Contacts

```python
import requests

# List all contacts with pagination
response = requests.get(
    "http://localhost:8882/api/v1/omni/my-whatsapp/contacts",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"page": 1, "page_size": 100}
)

contacts = response.json()["contacts"]
print(f"Found {len(contacts)} contacts")

# Search for specific contacts
response = requests.get(
    "http://localhost:8882/api/v1/omni/my-whatsapp/contacts",
    headers={"x-api-key": "YOUR_API_KEY"},
    params={"search_query": "John", "page_size": 50}
)

matches = response.json()["contacts"]
for contact in matches:
    print(f"{contact['display_name']}: {contact['phone_number']}")
```

### Get Contact Details

```python
# Fetch specific contact by ID
contact_id = "5511999999999@s.whatsapp.net"
response = requests.get(
    f"http://localhost:8882/api/v1/omni/my-whatsapp/contacts/{contact_id}",
    headers={"x-api-key": "YOUR_API_KEY"}
)

if response.status_code == 200:
    contact = response.json()
    print(f"Contact: {contact['display_name']}")
    print(f"Status: {contact['status']}")
    print(f"Last seen: {contact['last_seen']}")
```

### Sync Contacts Across Channels

```python
# Get contacts from WhatsApp instance
whatsapp_contacts = requests.get(
    "http://localhost:8882/api/v1/omni/my-whatsapp/contacts",
    headers={"x-api-key": "YOUR_API_KEY"}
).json()["contacts"]

# Get contacts from Discord instance
discord_contacts = requests.get(
    "http://localhost:8882/api/v1/omni/discord-bot/contacts",
    headers={"x-api-key": "YOUR_API_KEY"}
).json()["contacts"]

print(f"WhatsApp: {len(whatsapp_contacts)} contacts")
print(f"Discord: {len(discord_contacts)} contacts")
```
