### Step 2: Run Your First Agent

Open a NEW terminal (keep servers running):

```bash
genie run plan "Analyze the architecture of this codebase"
```

**Output:**
```json
{
  "sessionId": "abc-123-def",
  "status": "running",
  "agent": "plan",
  "executor": "claude-code",
  "forgeUrl": "http://localhost:8887/sessions/abc-123-def"
}
```

**What's happening:**
1. Genie CLI sends task to Forge backend
2. Forge executes using your chosen executor (Claude Code, OpenCode, etc.)
3. Results stream to Forge database
4. You get a session ID to track progress

**What each piece does:**

1. **Genie CLI** - Command-line interface you interact with
2. **Forge Backend** - Server that manages task execution (port 8887)

4. **Executor** - The AI that actually does the work (Claude, OpenCode, etc.)
5. **Database** - Stores sessions, results, and history

## The Two Modes

### Server Mode

```bash
genie  # Starts Forge + MCP, keeps running
```

**Use when:**
- Starting work on a project
- Need persistent backend
- Using IDE integration (Cursor, Claude Code)

**Stops when:**
- You press Ctrl+C
- You close the terminal
- System shuts down

### Command Mode

```bash
genie run plan "..."        # Execute agent, get result
genie talk architect        # Open browser session
genie list sessions         # Show active sessions
genie view <sessionId>      # View session results
genie stop <sessionId>      # Stop running session
```

**Use when:**
- Forge backend is already running (from server mode)
- Quick one-off commands
- Checking status or viewing results

## Troubleshooting

<AccordionGroup>
  <Accordion title="Forge won't start">
    **Symptom:**
    ```bash
    genie
    # âŒ Error: Failed to start Forge backend
    # Error: Port 8887 already in use
    ```

    **Fix:**
    ```bash
    # Check what's using port 8887
    lsof -i :8887
    # Or on Windows:
    netstat -ano | findstr :8887

    # Kill the process using that port
    kill -9 <PID>

    # Or use a different port
    FORGE_PORT=8888 genie
    ```

    **Check logs:**
    ```bash
    cat .genie/state/forge.log
    ```
  </Accordion>

  <Accordion title="No agents found">
    **Symptom:**
    ```bash
    genie list agents
    # âŒ No agents available
    ```

    **Fix:**
    ```bash
    # Reinitialize Genie
    genie init code --yes

    # Verify .genie directory exists
    ls -la .genie/

    # Check if templates were downloaded
    ls -la .genie/collectives/
    ```
  </Accordion>

  <Accordion title="Session not found">
    **Symptom:**
    ```bash
    genie view abc-123-def
    # âŒ Error: Session not found
    ```

    **Fix:**
    ```bash
    # List all sessions
    genie list sessions

    # Check if session ID is correct
    # Session IDs are case-sensitive

    # Check if Forge backend is running
    genie status
    ```
  </Accordion>

  <Accordion title="Command hangs / no response">
    **Symptom:**
    ```bash
    genie run plan "..."
    # (nothing happens)
    ```

    **Fix:**
    ```bash
    # 1. Check if Forge is running
    genie status

    # 2. Check Forge logs
    tail -f .genie/state/forge.log

    # 3. Check if executor has API key
    cat .genie/config.json | grep apiKey

    # 4. Test network connectivity
    curl http://localhost:8887/health
    # Should return: {"status":"ok"}

    # 5. Restart Forge
    # Kill genie server (Ctrl+C)
    # Start again: genie
    ```
  </Accordion>

  <Accordion title="API key errors">
    **Symptom:**
    ```bash
    genie run plan "..."
    # âŒ Error: Invalid API key
    ```

    **Fix:**
    ```bash
    # Set API key during init
    genie init code

    # Or configure manually
    # Edit .genie/config.json:
    {
      "executor": {
        "type": "claude-code",
        "apiKey": "sk-ant-your-key-here"
      }
    }

    # Or use environment variable
    export ANTHROPIC_API_KEY="sk-ant-..."
    genie run plan "..."
    ```
  </Accordion>

  <Accordion title="Browser doesn't open">
    **Symptom:**
    ```bash
    genie talk architect
    # (no browser window opens)
    ```

    **Fix:**
    ```bash
    # Manually open the URL
    # The command prints the URL - copy it
    genie talk architect
    # Open in browser: http://localhost:8887/chat/architect

    # Or check if default browser is set
    echo $BROWSER  # Linux/Mac
    ```
  </Accordion>
</AccordionGroup>
