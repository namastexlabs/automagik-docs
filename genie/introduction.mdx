## What Is Genie?

Genie is a CLI that orchestrates AI coding agents through Automagik Forge.

**Simple version:** It's a command-line interface that lets you run AI agents on your codebase.

## What It Does

1. **Manages agents** - 34 code agents + 9 content agents (markdown files with prompts)
2. **Executes via Forge** - Backend server that runs the actual agent work
3. **Provides MCP server** - So Claude Desktop can call Genie commands
4. **Tracks sessions** - All agent work stored and retrievable

## What It Doesn't Do

- ❌ Replace your coding workflow
- ❌ Work without Forge (it's a required dependency)
- ❌ Provide a terminal REPL for natural language
- ❌ Run agents locally (they use external APIs like Claude, GPT)

## Architecture

```
You
 ↓
genie CLI (this tool)
 ↓
Forge Backend (task orchestration)
 ↓
Executor (Claude Code, OpenCode, Codex, etc.)
 ↓
Your codebase
```

## Key Concepts

**Agents** - Markdown files with prompts (`.genie/agents/*.md`)
**Collectives** - Groups of agents (Code, Create)
**Spells** - Reusable knowledge patterns
**Forge** - Backend server (runs on port 8887)
**MCP** - Model Context Protocol server (runs on port 8885)

## Installation

Two dependencies:
1. Genie (this CLI)
2. Forge (backend server)

Both installed together:
```bash
npm install -g automagik-genie
```
