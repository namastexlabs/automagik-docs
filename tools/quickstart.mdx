## 2-Minute Example: Genie

Genie orchestrates MCP servers with persistent memory.

```bash
# Start Genie
uvx automagik-tools tool genie -t stdio
```

**Connect to Claude Desktop:**

Edit `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%\Claude\claude_desktop_config.json` (Windows):

```json
{
  "mcpServers": {
    "genie": {
      "command": "uvx",
      "args": ["automagik-tools", "tool", "genie", "-t", "stdio"],
      "env": {
        "OPENAI_API_KEY": "sk-your-key"
      }
    }
  }
}
```

Restart Claude Desktop. Test: "What MCP tools are available?"

## Example: Stripe API to MCP

```bash
# Download Stripe OpenAPI spec
curl -o stripe.json https://raw.githubusercontent.com/stripe/openapi/master/openapi/spec3.json

# Convert to MCP server
uvx automagik-tools openapi stripe.json -t stdio
```

**Configure in Claude Desktop:**

```json
{
  "mcpServers": {
    "stripe": {
      "command": "uvx",
      "args": ["automagik-tools", "openapi", "/path/to/stripe.json", "-t", "stdio"],
      "env": {"STRIPE_API_KEY": "sk_test_your_key"}
    }
  }
}
```

Ask Claude: "Create a $100 payment for customer cus_123"

Claude uses the Stripe MCP server. Processes payment. Returns result.

## Example: Genie with Multiple MCPs

Configure Genie to orchestrate multiple MCP servers:

```bash
export GENIE_MCP_CONFIGS='{
  "filesystem": {
    "command": "npx",
    "args": ["-y", "@modelcontextprotocol/server-filesystem", "/workspace"],
    "transport": "stdio"
  },
  "github": {
    "command": "uvx",
    "args": ["mcp-server-git"],
    "env": {"GITHUB_TOKEN": "your-token"},
    "transport": "stdio"
  }
}'

uvx automagik-tools tool genie -t stdio
```

Ask Claude: "Use memory to track my preferences, analyze GitHub repos, save summaries to filesystem"

Genie coordinates all three MCP servers. Maintains context. Executes workflow.

## Transport Modes

**stdio** - Local use only:
```bash
uvx automagik-tools tool genie -t stdio
```

**SSE** - Network access, team sharing:
```bash
uvx automagik-tools tool genie -t sse --port 8000
```

**HTTP** - REST API access:
```bash
uvx automagik-tools tool genie -t http --port 8000
```
