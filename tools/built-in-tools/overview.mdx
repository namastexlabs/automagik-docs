## The Tools

| Tool | What it does | Required env vars |
|------|-------------|-------------------|
| **genie** | Orchestrates other MCP servers with persistent memory | OPENAI_API_KEY |
| **automagik** | Integrates with Automagik Suite (Spark, Hive, Forge, Omni) | AUTOMAGIK_API_KEY, AUTOMAGIK_BASE_URL |
| **evolution-api** | WhatsApp automation via Evolution API v2 | EVOLUTION_API_KEY, EVOLUTION_API_BASE_URL |
| **omni** | Multi-channel messaging (WhatsApp, Slack, Discord) | OMNI_API_KEY |
| **spark** | Temporal automation and workflow scheduling | SPARK_API_KEY |
| **gemini-assistant** | Google Gemini AI integration | GOOGLE_API_KEY |
| **automagik-workflows** | Execute Claude Code workflows programmatically | None (uses local files) |
| **automagik-hive** | Test Automagik Hive API | AUTOMAGIK_HIVE_API_KEY |
| **wait** | Simple wait functionality for workflows | None |

## Start a Tool

All tools use the same command format:

```bash
# For Claude Desktop (stdio)
uvx automagik-tools tool TOOLNAME -t stdio

# For team sharing (SSE)
uvx automagik-tools tool TOOLNAME -t sse --port 8000

# For HTTP API
uvx automagik-tools tool TOOLNAME -t http --port 8080
```

When you run it, you see:
```
INFO Starting MCP server 'TOOLNAME' with transport 'stdio'
```

The server runs until you stop it (Ctrl+C).

## Claude Desktop Integration

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "genie": {
      "command": "uvx",
      "args": ["automagik-tools", "tool", "genie", "-t", "stdio"]
    },
    "evolution-api": {
      "command": "uvx",
      "args": ["automagik-tools", "tool", "evolution-api", "-t", "stdio"],
      "env": {
        "EVOLUTION_API_KEY": "your-key",
        "EVOLUTION_API_BASE_URL": "https://your-instance.com"
      }
    }
  }
}
```

Restart Claude Desktop. Ask: "what tools are available?"

## Common Combinations

### AI-powered WhatsApp bot
```bash
# Terminal 1: Start Genie
uvx automagik-tools tool genie -t sse --port 8000

# Terminal 2: Start Evolution API
uvx automagik-tools tool evolution-api -t stdio
```

Genie coordinates Evolution API for intelligent responses.

### Multi-channel notifications
```bash
# Start Omni for all platforms
uvx automagik-tools tool omni -t stdio
```

Send to WhatsApp, Slack, Discord with one command.

### Scheduled automation
```bash
# Start Spark for cron jobs
uvx automagik-tools tool spark -t stdio
```

Schedule recurring tasks and workflows.

## Troubleshooting

**Tool won't start**:
```bash
# Check if already running
ps aux | grep automagik-tools

# Check port availability
lsof -i :8000

# Try different port
uvx automagik-tools tool genie -t sse --port 8001
```

**Claude can't see tool**:
1. Restart Claude Desktop
2. Check config file syntax (valid JSON)
3. Verify tool is running: `ps aux | grep TOOLNAME`
4. Check Claude logs

**Missing environment variables**:
```bash
# Check what's required
uvx automagik-tools info TOOLNAME

# Set in shell or config file
export TOOL_API_KEY="your-key"
```
