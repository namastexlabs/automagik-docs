## Install

<Tabs>
  <Tab title="uvx (Recommended)">
    ```bash
    # Install uv
    curl -LsSf https://astral.sh/uv/install.sh | sh

    # Use immediately, no installation needed
    uvx automagik-tools list
    ```

    No global installation. No virtual environments. Just works.
  </Tab>

  <Tab title="pip">
    ```bash
    pip install automagik-tools
    automagik-tools list
    ```
  </Tab>

  <Tab title="pipx">
    ```bash
    pipx install automagik-tools
    automagik-tools list
    ```
  </Tab>
</Tabs>

## Verify

```bash
uvx automagik-tools --help
```

Output shows these commands:
- `list` - List all available tools
- `hub` - Serve all tools with path-based routing
- `tool` - Serve a specific tool
- `openapi` - Serve tool from OpenAPI spec URL
- `info` - Show detailed information about a tool
- `create-tool` - Create new MCP tool from OpenAPI spec
- `mcp-config` - Generate MCP configuration for Cursor/Claude
- `version` - Show version information

## API Keys (Optional)

Required only for specific tools:

**Genie:**
```bash
export OPENAI_API_KEY=sk-your-key
```

**Evolution API:**
```bash
export EVOLUTION_API_KEY=your-key
export EVOLUTION_API_BASE_URL=https://your-instance.com
```

**Automagik Suite:**
```bash
export AUTOMAGIK_API_KEY=your-key
export AUTOMAGIK_BASE_URL=http://localhost:8881
```

Store in `.env` file or export before running commands.
