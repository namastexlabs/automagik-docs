**Production-Ready AI Teams in 5 Minutes**

YAML-based agent creation with hot-reload, persistent memory, and RAG built-in. One-click install with complete stack included.

### Key Features

- **YAML Configuration**: Define agents with simple, declarative syntax
- **Hot-Reload**: Instant updates without restarting your agents
- **Persistent Memory**: Agents remember context across sessions
- **Built-in RAG**: Retrieval-Augmented Generation out of the box
- **Complete Stack**: Everything you need included in one package
