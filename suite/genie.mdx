**The Glue Between AI Agents and Repositories**

Ships ready-to-run AI workspace in minutes, keeps templates in sync, exposes battle-tested CLI + MCP server for any agent to work with context.

### Key Features

- **Instant Workspace**: Ready-to-run AI environment in minutes
- **Template Sync**: Keep your project templates up to date
- **Battle-Tested CLI**: Production-ready command-line interface
- **MCP Server**: Standards-based agent integration
- **Context-Aware**: Deep repository understanding for AI agents
